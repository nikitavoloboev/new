# New

> Start new projects, create files quickly

Templates will be built using [template CLI](https://github.com/fabiospampinato/template) and have instructions inside.

There is old starter [here](https://github.com/nikitavoloboev/past/tree/main/april/new-solid-grafbase-edgedb-tauri) but its very out of date.

New stack can be seen in [KusKus](https://github.com/kuskusapp/kuskus) & [LA](https://github.com/learn-anything/learn-anything.xyz).

Currently [mobile](mobile) can be served as base to start new RN/Expo projects. It's stripped down version of [React Conf app](https://github.com/expo/react-conf-app).
